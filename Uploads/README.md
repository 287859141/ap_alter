系统结构
www  WEB部署目录（或者子目录）
├─index.php       入口文件
├─.htaccess       htaccess 
├─README.md       README文件
├─Application     应用目录
├─Public          资源文件目录
├─TP              框架目录
├─.settings       框架目录
├─Common          公共类库
├─Public          公共模型、方法、HTML、CSS、JS
└─Uploads         上传文件
thinkPHP目录结构（Tp）
├─ThinkPHP 框架系统目录（可以部署在非web目录下面）
│  ├─Common       核心公共函数目录
│  ├─Conf         核心配置目录 
│  ├─Lang         核心语言包目录
│  ├─Library      框架类库目录
│  │  ├─Think     核心Think类库包目录
│  │  ├─Behavior  行为类库目录
│  │  ├─Org       Org类库包目录
│  │  ├─Vendor    第三方类库目录
│  │  ├─ ...      更多类库目录
│  ├─Mode         框架应用模式目录
│  ├─Tpl          系统模板目录
│  ├─LICENSE.txt  框架授权协议文件
│  ├─logo.png     框架LOGO文件
│  ├─README.txt   框架README文件
└─ └─ThinkPHP.php 框架入口文件
